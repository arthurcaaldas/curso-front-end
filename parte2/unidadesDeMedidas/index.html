Unidade de medidas

em/rem

tamanho da pagina difinido em: 62.5%; é a mesma coisa q 1em = 10px

1em; = 16px;

1rem; = se baseia no tamanho definido pelo html (pode ser alterado);

Porcentagem em margin

vai se basear no tamanho definido no html ou no pai

vw e vh

90vw é mesma coisa que 90% para width (largura) 90vh é mesma coisa que 90% para height (altura)

ch

usada para definir a quantidade de caracteres por linha

Função calc()

trabalha com operações mat para ter uma maior precisao no tamanho da tela. Ex: min-height: calc(100vh - 15rem - 2rem);

variaveis css

Declarando variavel: --header-height: 10rem; Usando a variavel: height: var(--header-height: 10rem;); define valores por variaveis (vantagem: consegue mudar tudo de uma vez só);

outline

border cria uma borda q faz parte do conteudo

não altera as propriedades do conteudo original
